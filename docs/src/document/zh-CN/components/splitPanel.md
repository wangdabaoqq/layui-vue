::: title splitPanel属性
:::

::: table

| 属性  | 描述 | 类型 |可选值 | 默认值|
| ----- | ---- | ------ | ---| ---|
| vertical | 是否垂直布局 | Boolean |`true` `false`| false |
| minSize | 块拉动最小范围(按像素 `px`) | number | - | 50  |
:::

::: title splitPanelItem属性
:::

::: table

| 属性  | 描述 | 类型 |可选值 | 默认值|
| ----- | ---- | ------ | ---| ---|
| space | 设置当前item宽度，可传固定宽度如 `300` 、 `300px` 、 `20%`。必须传数值类型，不可传 如 `auto` 值 | `number` | `string` | - | 按照个数平分 |
:::
